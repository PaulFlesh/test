Комплектующие для производства пластиковых окон, стеклопакетов

Угол соединительный для дистанционной рамки.

Дистанционная рамка при сборке стеклопакета соединяется угловыми соединениями (уголками) соответствующей ширины. Упаковка готовой продукции производиться в картонные короба. Норма фасовки уголков приведена в таблице.

Фальцевый вкладыш 171

Устанавливается в раму или створку окна (под стеклопакет), чтобы равномерно распределить нагрузку стеклопакета на конструкцию. Фальцевый вкладыш КВЕ 171 подходит для профилей КВЕ,EXPROF,PROPLEX шириной 58 мм.

Комплектующие для москитных сеток.

Угол соединительный цельнолитой- Цвет белый АБС. Угол соединительный пластиковый предназначен для соединения рамных профилей под прямым углом.

Крепление верх-низ (пара)-цвет белый. АБС. Крепление предназначено для фиксации на окне москитной сетки. Изделие вставляется с наружной стороны и фиксируется креплениями по углам проема.

Ручка «бабочка» - цвет белый. АБС. Ручка крепится на профиле москитной сетки шнуром и используется для установки москитной сетки на окна.

Крепление поперечины - цвет белый. АБС. Крепление поперечины под шнур, предназначен для соединения поперечного профиля с пазами под шнур с рамным профилем.
